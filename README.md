# parrot-care-agent
牡丹鹦鹉饲养过程中，需要实时关注环境温湿度、光照、饮食状态，以及鹦鹉的行为异常（如拒食、过度活跃/萎靡），人工管理耗时且易遗漏。本项目基于正点原子 RK3588 开发板，结合嵌入式开发、机器视觉、传感器采集技术，打造一款轻量化的智能管理智能体，实现饲养过程的半自动化与智能化。
# 牡丹鹦鹉日常管理智能体
基于正点原子 RK3588 开发板的鹦鹉饲养智能管理系统，实现环境监测、行为识别、自动喂食等功能。

## 硬件平台
- 主控：正点原子 RK3588 开发板
- 外设：摄像头、温湿度传感器、光照传感器、舵机等

## 功能规划
- [x] 环境数据采集（温湿度、光照）
- [x] 摄像头实时画面采集
- [ ] 鹦鹉行为识别与异常检测
- [ ] 自动定时喂食/喂水
- [ ] 数据上传与远程查看
